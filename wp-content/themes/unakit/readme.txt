== Description ==

Unakit is a modern theme for professional websites. It’s fully responsive and looks beautiful on every screen. Focusing on maximizing browser compatibility, it allows a consistent and reliable presentation to be easily achieved. With a minimalistic design and commitment to accessibility, every visitor can easily understand your content and receive your message. Build confidence in your website and performance using this heavily optimized and inherently private theme. Create content quickly and flexibly with the Gutenberg editor and seamlessly integrate your website’s appearance into your brand using numerous customization options. Start building your professional website now with Unakit!

== Privacy Policy ==

Unakit uses Freemius (freemius.com) as a platform for license management and software updates. If you opt in to providing data, diagnostic data will be collected by Freemius Inc in order to manage licenses and updates if applicable, as well as providing insights about the usage of the theme that allow to spot issues with the theme or conflicts with other themes or plugins. Diagnostic data may include:
* Site URL
* WordPress version
* PHP info
* Current theme status (active, inactive, uninstalled)
* Plugins and themes

Opting in is purely voluntary and not required for any features of the free version of the theme.
Please see the privacy policy of Freemius at https://freemius.com/privacy/ for further details.

Unakit offers fonts initially provided by Google Fonts. However, for increased privacy and reliability, it serves them locally, bundled with the theme, and does not create a connection to Google's servers.
